# Student-Record-Management-System

- This project demonstrates persistent student data management using **file handling** and **Java Collection Framework** with sorting and searching features.

---

## Learning Goals
- Read/write student data using `BufferedReader` & `BufferedWriter`
- Maintain records in `ArrayList`
- Sort using `Comparator` and `Comparable`
- Display results using `Iterator`
- Provide a menu-driven console application

---

## Folder Structure
- src/
- ├── entity/Student.java
- ├── io/FileHandler.java
- ├── controller/StudentRecordSystem.java
- └── App.java


---

## Features
- Load records at startup  
- Register new students  
- Search & delete by name  
- Sort by marks (highest first)  
- Save updated records on exit  
- Uses `Iterator` for display
